Readme for VAST data generation locata challenge

Locally:

1. run the "DemoVAST.m" file in the "mFiles" folder
2. get your results in the "Results" folder

On Igrida:
1. Copy all the directories and files keeping the tree organization
2. Compile the code running "run_IGRIDA_compilation.sh"
3. Make sure that in the "bin" folder you have:
  - Absorption.mat
  - DataSetGeneration_launcher.sh
  - roomsim.mexa64

  and:
  - run_TestSubVASTGeneration.sh
  - TestSubVASTGeneration
  - VASTGeneration.sh

4. To launch the generation run "DataSetGeneration_launcher.sh"
5. Get your results in the "Results" folder
